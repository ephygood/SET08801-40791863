About This Project

This project simulates the monitoring of ATM devices. The user is prompted to click on an ATM device to view whether that device is online or offline.

The site uses SVG status images, click sound effects, reusable JavaScript functions, and an external CSS stylesheet to keep the project organised.

Technologies Used

  • HTML for the page structure
  • External CSS for styling
  • JavaScript for status checking
  • SVG images for visual status feedback
  • Audio feedback for ATM click interaction
